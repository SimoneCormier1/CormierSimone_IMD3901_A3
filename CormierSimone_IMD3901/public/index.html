Welcome to the Color Picker Game!

Choose below whether you want 2D or 3D mode

1 player must be in 3D, the other player must be in 2D

3D player starts the game by pressing Start button. 3D player must choose 4 colors, and then it is 2D players turn. 2D player must choose the same 4 colors as 3D chose to earn points. Then same thing but 2D player starts. First to 50 points wins! Be careful, you must work together to finish before 100 seconds or you both lose!
